# Khent Andrei Tenchavez — Portfolio Site

Fully responsive (desktop, tablet, mobile) single-page portfolio.

## Files
- `index.html` — the whole site (HTML, CSS, JS in one file)
- `images/kenpfp.png` — profile photo (already included)
- `images/klogo1.png` — logo mark used in the navbar (already included)

## Deploy (free, no coding)
1. Netlify Drop: https://app.netlify.com/drop — drag this whole folder in, done.
2. Vercel: https://vercel.com → Add New Project → Upload this folder.
3. GitHub Pages: push `index.html` + `images/` to a repo, enable Pages in Settings.
4. Or just double-click `index.html` to preview locally — works offline.
